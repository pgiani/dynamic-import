### Get it

```sh
git clone https://github.com/pgiani/dynamic-import
cd dynamic-import
```

Checkout this repo to the appropriate branch:

```sh
git checkout 01-initial
git checkout 02-problem

```

Be sure to install `npm` packages for your branch:

```sh
meteor npm install
```

```sh
npm start
```
